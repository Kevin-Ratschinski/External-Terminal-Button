# Changelog

All notable changes to the "External Terminal Button" extension will be documented in this file.

## [Unreleased]

## [1.2.0] - 2026-03-28

### Added

- New setting `terminalApp` to specify a custom terminal application (e.g., Ghostty, iTerm, Warp, Windows Terminal)

## [1.1.0] - 2026-03-23

### Added

- New setting `openInFileDirectory` to open the terminal in the directory of the currently active file

## [1.0.0] - 2026-03-16

### Added

- Status bar button to open an external terminal
- Configurable button text, alignment, and priority via settings
- Live config update without reload
